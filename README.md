# Aynkaran Astro Center

A professional astrology consultation website built with React, Tailwind CSS, and Framer Motion.

## Features

- ✨ Smooth scroll animations and transitions
- 🎨 Modern glassmorphism UI design
- 📱 Fully responsive design
- 🌟 Interactive zodiac horoscope system
- 📅 Multi-step booking consultation form
- ⚡ Fast performance with Vite
- 🎯 SEO optimized

## Tech Stack

- React 18
- React Router DOM
- Framer Motion (animations)
- GSAP (advanced animations)
- Tailwind CSS
- Lucide React (icons)
- Vite (build tool)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
